# Petals Magazine V4

Bold editorial redesign for **MG International Fragrance Company**'s digital publication.

Design DNA: i-D × Dazed × Vogue — full-bleed imagery, asymmetric grids, serif dominance, monochromatic palette.

## Setup

```bash
npm install
npm run dev
```

## Deploy to Vercel

```bash
npx vercel
```

## Stack

- React 18
- Vite 6
- Google Fonts (Bodoni Moda + DM Sans)
- Unsplash imagery
